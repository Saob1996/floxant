# FLOXANT Platform

Premium Umzugs- und Service-Betriebssystem für Regensburg und Bayern.

## Projektstatus: KONSOLIDIERT / BETRIEBSMODUS ✅

FLOXANT ist nach Abschluss der Härtungsphasen und der gezielten Regionalexpansion (Stand 04/2026) vollständig operativ einsatzbereit. Das System umfasst eine souveräne öffentliche Website mit regionalen Fokus-Hubs, einen intelligenten Intake-Rechner sowie ein vollumfängliches Backoffice-System.

## Technische Basis
- **Stack**: Next.js 16 (Turbopack), TailwindCSS, TypeScript.
- **Backend**: Supabase (PostgreSQL), Resend (Mailing).
- **Infrastruktur**: Gehärtete API-Routen mit Informations-Leaking-Schutz.
