# FLOXANT System Handover

Offizielle Übergabe der FLOXANT Plattform an den operativen Bereich.

## 1. Systemstatus
- **Öffentliche Website**: 100% produktiv. Alle Routen (Home, /rechner, City-Pages, Regional-Expansion) aktiv.
- **Intake-System**: Drei-Service-Logik (Umzug, Reinigung, Entsorgung) vollständig operativ.
- **Regionaler Ausbau**: 8 priorisierte Landingpages für Regensburg und Bayern aktiv (Stand 04/2026).
- **Backoffice**: Dashboard mit Lead-Management, Dokument-Tracker und Health-Monitoring.
- **Dokumentensystem**: Snapshot-basierte Erzeugung von Angeboten, ABs und Rechnungen.

## 2. Technische Produktionsbasis
- **Hosting**: Node.js Produktion (Turbopack Build).
- **Datenbank**: Supabase (PostgreSQL) mit RLS-Schutzregeln.
- **E-Mail**: Resend API Integration (Live-Modus aktiv bei gesetztem API-Key).
- **Zustandsüberwachung**: AdminHealthControl Widget zur Verifizierung der DB- und Mail-Integrität.

## 4. Aktuelle Expansion (April 2026)
Die Phase "Selective Regional Expansion" wurde erfolgreich abgeschlossen. 
- **Fokus**: Regensburg (Welterbestadt & Umland) und Bayern (B2B Hub).
- **Inhalt**: 7 hochspezialisierte Seiten für Regensburg, 1 Bayern-B2B Anker.
- **Design**: Premium Dark Surface Standard vollständig ausgerollt.
