## Signature Services

| Service | Aufgabe | Grenzen |
| --- | --- | --- |
| FLOXANT Angebotscheck | Angebot, Umfang und offene Punkte pruefen | keine Rechtsberatung, keine Preisgarantie |
| FLOXANT Fairpreis-Check | Aufwandstreiber und Preislogik einordnen | keine Unterbietungszusage |
| FLOXANT Objektbrief | Objekt, Fotos, Zugang und Zielzustand strukturieren | keine automatische Zusage |
| FLOXANT Uebergabeakte | Fotos, Restpunkte und Schluesselweg buendeln | keine Abnahmegarantie |
| FLOXANT Uebergabe-Sprint | Frist, Restmengen und Reinigung priorisieren | keine Sofortgarantie |
| FLOXANT Plan-B-Service | Anbieterausfall oder Deadline sortieren | keine Kapazitaetsgarantie |
| FLOXANT Rueckfahrt-Radar | Route, Beiladung und Zeitfenster pruefen | keine erfundene Rueckfahrt |
| FLOXANT Diskret-Service | sensible Faelle ruhig anfragen | keine Rechts-/Sicherheitsberatung |
| FLOXANT Vermieter-Ready-Service | Objekt fuer Uebergabe/Nachnutzung vorbereiten | keine mietrechtliche Bewertung |
| FLOXANT Buero-Startklar-Service | Gewerbeflaechen startklar machen | keine Spezialzertifikate behaupten |
| FLOXANT PV-Sichtklar-Service | PV-Fotos, Zugang und Grenzen pruefen | keine Ertrags-, Dach- oder Elektrogarantie |
| FLOXANT Entscheidungs-Kompass | passenden Startpunkt finden | keine automatische Entscheidung ohne Rueckfragen |

## Interne Verlinkung

- `/signature-services` verweist auf Angebotscheck, Objektbrief, Plan B, Rueckfahrt, PV und Diskret.
- `/spezialreinigung` verknuepft PV, Glas, Fassade, Praxis, Hotel, Hausverwaltung und Angebotspruefung.
- `/spezialumzug` verknuepft Mini-Umzug, Express, Moebeltransport, Rueckfahrt und Plan B.
- `/spezial-entruempelung` verknuepft Keller, Lager, Nachlass, Wohnungsaufloesung, Uebergabe und Diskret.

## Risiken

- Einige Detailservices haben keine eigene bestaetigte Verfuegbarkeit je Standort.
- Keine neuen Routen wurden erzeugt; einige Links verweisen auf bereits vorhandene oder strategisch vorhandene Hubs.
- Preview muss pruefen, ob alle Zielrouten im aktuellen Worktree wirklich gebaut werden.
